005 — PROMPT ENGINEERING

Prompting Experiments

Real prompt patterns I've developed and tested to build smarter AI interactions.

Role-Based Prompting
SYSTEM PROMPT
"You are a friendly Tamil-speaking study buddy. When a student asks about NCERT topics, explain them using simple Tanglish (Tamil + English mix). Use real-life examples from Chennai or Tamil Nadu. Keep it conversational, fun, and easy to understand."
✅ Result: Increased student engagement by making responses feel culturally familiar
Chain-of-Thought
COT PROMPT
"Think step by step: First identify what concept the student is asking about, then check which NCERT class it belongs to, then break the concept into 3 simple parts, and finally give an example in Hinglish."
✅ Result: More structured and accurate educational explanations
Few-Shot Learning
FEW-SHOT PROMPT
Q: "Photosynthesis kya hota hai?"
A: "Bhai, photosynthesis matlab plants ka khana banana! Sun ki light leke, CO2 aur water se glucose banate hain..."

Q: "Osmosis explain karo"
A: [Model continues the pattern]
✅ Result: Consistent tone and language style across all responses
Constraint-Based
GUARD RAILS
"You must ONLY answer questions related to NCERT syllabus (Classes 6-12). If asked anything outside this scope, politely redirect: 'Dei/Bhai, that's outside my NCERT zone! Ask me about Science, Math, or Social Studies.'"
✅ Result: Focused AI that stays on-topic for educational use
